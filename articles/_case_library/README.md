# 案例库 (Case Library)

这个目录用于积累可复用的写作案例素材，按类型分类存储。

## 目录结构

```
_case_library/
├── README.md              # 本文件
├── 职场故事/              # 职场相关的故事案例
│   └── _template.md       # 模板
├── 认知反转/              # 认知颠覆类案例
│   └── _template.md       # 模板
└── 情绪共鸣/              # 情绪共鸣类案例
    └── _template.md       # 模板
```

## 使用方法

1. **积累案例**：写作过程中遇到好的案例，填写模板保存到对应分类
2. **查找案例**：写作时，先查看案例库是否有可用素材
3. **复用案例**：直接复制案例骨架，根据文章需要调整细节

## 案例模板格式

每个案例文件应包含：
- **故事骨架**：人物-冲突-转折-结果
- **可引用金句**：从案例中提炼的核心句子
- **适用主题**：这个案例适合用于哪些话题
- **读者预期反应**：读者看到这个案例会产生什么心理反应
